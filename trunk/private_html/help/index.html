Using the Admin System

To use the admin system you need to login. To login type in your username and password. Both are case sensitive

Once logged in you'll be at the main admin page. Here you'll see options to edit Users, Stories and access the Undo Buffer. You can logout at any time using the logout link on the top right. You can also access these help pages at any time by clicking help

Listing Data

By going to the Stories or Users section you'll be presented with a list of all the stories or users in the system. Some data, such as the content for a story, may not be listed in full here. Click edit to see the full data. Also some data, such as last login, cannot be edited

Adding Data

You can add more data by clicking 'Add New Record' on the Stories page to add a new story. Or on the Users page to add a new user. When presented with the form to add a new record you an cancel this action by clicking any link to any other section.

Editing Data

You can edit data by clicking 'Edit' next to the record you wish to edit in the Stories or Users page. When presented with the form to edit a record you an cancel this action by clicking any link to any other section.

Editing Data

You can delete data by clicking 'Delete' next to the record you wish to edit in the Stories. Users cannot be deleted and it is recommended you change the password for inactive users. You will be presented with a form to confirm the deletion, you can cancel by clicking any other link.

The Undo Buffer

The undo buffer lets you undo any actions taken within the last 7 days. This will undo any add, edit or delete action. You can only undo actions in sequence with the most recent being undone first. You cannot redo an action using the undo buffer
